### Para Executar o Código

- Não se esqueça de colocar o .csv na pasta
- Não se esqueça de baixar todas as dependencias

Execute primeiro find_cols.py e find_cols2.py

Depois rode o analysis.py

Depois use esse comando:
```python -m http.server 8000```

E depois cole no seu navegador: "<http://localhost:8000/index.html>"
